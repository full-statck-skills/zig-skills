# Zig 0.16 Examples

Offline example entry points for the `zig-0.16` skill.

## Files

- `quickstart-workflows.md`: starter build, test, JSON, process, HTTP, and review examples.
- `build-zig-zon-workflows.md`: package metadata, dependency wiring, executable and library layouts.
- `comptime-patterns.md`: reflection, generated types, inline loops, and compile-time validation.
- `c-interop-workflows.md`: header import, C source integration, exported symbols, and ABI notes.
- `std-thread-patterns.md`: spawn and join, mutex, wait group, and atomic counter patterns.

## Suggested usage

- Read this directory first when you need copyable snippets quickly.
- Use `../SKILL.md` for the version-specific migration notes and official coverage map.
- Use `../references/*.md` when you need deeper module-level detail.
