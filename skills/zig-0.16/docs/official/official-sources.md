# Official Zig 0.16 Sources (Offline Distillation)

This directory contains the offline distillations of the official Zig 0.16.0 docs and the Chinese homepage. It is separated from `references/` so that `references/` can stay focused on topic cards such as `language.md`, `builtins.md`, and `std-*.md`.

## Source pages

- Language reference: https://ziglang.org/documentation/0.16.0/
- Introduction section: https://ziglang.org/documentation/0.16.0/#Introduction
- Standard library index: https://ziglang.org/documentation/0.16.0/std/
- Chinese homepage: https://ziglang.org/zh-CN/

## Offline distilled files in this folder

- `official-language-reference-0.16.md`
- `official-introduction-0.16.md`
- `official-std-index-0.16.md`
- `official-zh-cn-home-0.16.md`

## How to use with the rest of the skill

- Use `../../references/language.md` + `../../references/patterns.md` when you need practical code guidance quickly.
- Use `../../references/builtins.md` as the offline reference for builtin functions referenced by the language docs.
- Use `../../references/std-*.md` module notes as the offline entry points for std modules referenced by the std index.
